Fast forward tracking branch to latest remote commit

I often find myself doing:
    
        git fetch --all
   
Then maybe at some later time, maybe without network:
    
        git co main-master
        git merge upstream-rw/master
    
The command `git-tff` below, allows me to shorten the last bit and make it less error prone, and general for any branch / remote tracking branch pair, as in:
    
        git co main-master
        git tff
